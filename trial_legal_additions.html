4. JavaScript — Enable Button on Checkbox

Add this JavaScript before the closing </body> tag to wire up the checkbox-to-button logic.

<script>
  // Enable/disable submit button based on checkbox state
  const termsCheckbox = document.getElementById('termsCheckbox');
  const submitBtn = document.getElementById('submitBtn');

  termsCheckbox.addEventListener('change', function() {
    submitBtn.disabled = !this.checked;
  });
</script>
How it works: When the checkbox is checked, the button becomes enabled. When unchecked, disabled again. Button starts disabled on page load.
