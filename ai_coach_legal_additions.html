AI Coach Legal Additions Patch

The Evolved Caveman Inner Leadership AI Coach — Compliance Update

Overview

This patch adds two legal compliance features to ai_coach.html:

  1. TOS Acceptance on Signup — Checkbox accepting Terms of Service, Privacy Policy, and AI Disclosure with linked documents
  2. Per-Session AI Disclosure Banner — Auto-dismissing banner reminding users they're interacting with AI (SB 243 compliance)

ADDITION 1: Terms of Service Acceptance (Signup Form)

Placement in Current Code

This addition goes in the signup form section, specifically:

<!-- ADD THIS AFTER the signup password input field -->
<!-- ADD THIS BEFORE the "Create Account" button -->

Current Code Reference

Find this section in your signup form:

<input type="password" id="signupPassword" placeholder="Password" required > <button id="createAccountBtn" >Create Account</button>

The checkbox and agreement section goes between these two elements.

JavaScript to Add (Enable/Disable Button)

// TOS CHECKBOX LOGIC const tosCheckbox = document.getElementById('tosCheckbox'); const createAccountBtn = document.getElementById('createAccountBtn'); // Initialize button as disabled createAccountBtn.disabled = true; // Toggle button state when checkbox changes tosCheckbox.addEventListener('change', function() { createAccountBtn.disabled = !this.checked; }); // IMPORTANT: Add this validation to your existing signup handler // (update your existing createAccount() or signup submit handler) // Before processing signup, verify checkbox: if (!tosCheckbox.checked) { console.error('User must accept TOS before signup'); alert('Please accept the Terms of Service, Privacy Policy, and AI Disclosure to continue.'); return false; }
Integration Note: Find your existing signup form submit handler (likely in a function like createAccount() or form submit event). Add the checkbox validation check above before calling your API/signup function. This ensures the agreement is accepted before account creation is processed.

ADDITION 2: Per-Session AI Disclosure Banner

Placement in Current Code

This addition is a new DOM element that appears in the chat area. Specifically:

<!-- ADD THIS AT THE TOP OF THE CHAT CONTAINER -->
<!-- Just after opening the chat messages area, before any existing chat content -->

Current Code Reference

Find your chat container section (likely looks like this):

<div id="chatContainer" class="chat-container"> <div id="chatHeader" >...</div> <div id="messages" class="messages"> <!-- messages go here --> </div>

The banner element goes either right after the chatHeader or at the top of the messages div.

HTML to Add (Banner Element)

<!-- AI DISCLOSURE BANNER (Dismissible) --> <div id="aiDisclosureBanner" class="ai-disclosure-banner" role="status" aria-live="polite" aria-label="AI disclosure notification" > <div class="banner-content"> <span class="banner-text"> Reminder: You are interacting with an AI system, not a human. JavaScript to Add (Banner Logic)

// AI DISCLOSURE BANNER LOGIC // Function to show and auto-dismiss the banner function showAIDisclosureBanner() { const banner = document.getElementById('aiDisclosureBanner'); const dismissBtn = document.getElementById('dismissBannerBtn'); // Ensure banner is visible banner.classList.remove('hidden'); // Auto-dismiss after 10 seconds (10000 ms) const autoDismissTimer = setTimeout(() => { banner.classList.add('hidden'); }, 10000); // Manual dismiss button handler dismissBtn.addEventListener('click', function() { clearTimeout(autoDismissTimer); banner.classList.add('hidden'); }); } // IMPORTANT: Call this function when user enters chat // Find your enterChat() function and add this line: // showAIDisclosureBanner(); // Example: if your function is called when entering chat: // function enterChat() { // ... existing code ... // showAIDisclosureBanner(); // ADD THIS LINE // }
Critical Integration Point: The banner must be shown each time a new session starts. Find your enterChat() function or equivalent (the function that transitions from login to chat view). Add showAIDisclosureBanner(); to display the banner when chat begins.
